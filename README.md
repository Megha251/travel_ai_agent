# ✈️ AI Travel Planning Assistant

## 📌 Project Overview

AI Travel Planning Assistant is a Python-based travel planner that helps users generate travel plans automatically using JSON datasets and live weather data.

The system provides:

- Flight recommendations
- Hotel suggestions
- Tourist attractions
- Weather updates
- Budget estimation
- Day-wise itinerary planning

The application is built using:

- Python
- Streamlit
- LangChain
- Open-Meteo API

# ⚙️ Installation

## Step 1: Clone Project

```bash
git clone https://github.com/Megha251/travel_ai_agent.git
```

---

## Step 2: Create Virtual Environment

```bash
python -m venv venv
```

---

## Step 3: Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

---

# 📦 Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Streamlit Application

```bash
streamlit run app.py
```
